# Configuration of the Logre environment

## The base URI for instances

https://borders.lod4hss.org/resource/

In Logre, all URI already starts with an "i" (in other platforms, such as WissKI, it is needed to add an "i").

## The various URI of the named graphs

The named graph for the information (as opposed to factoïds, models, etc.) should be:
https://borders.lod4hss.org/graph/information

The named graph for the model, in the form of SHACL profiles, should be:
https://borders.lod4hss.org/graph/shacl

The named graph for the graph metadata (for the moment not used) should be:
https://borders.lod4hss.org/graph/metadata
